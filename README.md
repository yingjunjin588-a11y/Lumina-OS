# Lumina-OS
一个自制的scratch OS | A homemade Scratch OS

由个人独立开发的类 Scratch 桌面操作系统，主打轻量化、可定制化，兼顾开发者调试体验与普通用户使用便捷性。

## 📖 项目简介
Lumina OS 是一款面向创意开发与轻量办公的自制操作系统，核心设计理念为「简洁而不简单」：
- 保留 Scratch 风格的可视化操作逻辑，降低使用门槛
- 内置完善的调试体系，支持开发者自定义 Debug 提示

## ⚙️ 开发说明
- 本项目为个人独立开发，持续迭代中
- 定位：轻量级创意开发操作系统，非生产级系统
- 调试模式：默认开启 Debug 提示
